# Client Feedback Management System - Backend

This is the backend for the Client Feedback Management System.

## Features
- User registration/login (JWT)
- Public feedback submission
- Admin routes to view, update, and delete feedback

## Run locally
1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies: `npm install` in the `backend` folder.
3. Start server: `npm run dev` (requires nodemon) or `npm start`.
